# Local-ChatBot
Local Chatbot running on LLama 3.1, coded with google Antigravity IDE.
If you choose to use the script you will need to set up a local Anaconda environment and get all required packages.
The code was developed with a Windows 11 O.S running an Nvidia RTX5070 Ti Laptop, AMD Ryzen 9 8940HX CPU and 32.0 GB RAM capacity.
